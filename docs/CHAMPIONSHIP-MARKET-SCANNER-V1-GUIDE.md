# Championship Market Scanner v1

## Install
```cmd
pip install -r requirements_championship_scanner_v1.txt
```

## Offline test
```cmd
python test_championship_market_scanner_v1.py
```

## Download scanner data
```cmd
python download_scanner_market_data.py
```

## Run scanner
```cmd
python championship_market_scanner_v1.py --data-dir data\scanner
```

## Reports
- reports\scanner\championship_scanner_v1.json
- reports\scanner\top_longs.csv
- reports\scanner\top_shorts.csv
- reports\scanner\championship_scanner_v1_summary.txt

Research only. Short availability, borrow costs, earnings, news, and intraday execution are not yet modeled.
